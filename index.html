what's codeMath?

codeMath is an initiative that seeks to explore new ways of teaching Math through Code. It seeks to find new ways of approaching two topics that throughout time have been proved to be hard when learned.

trajectory

This is a long term project with different stages. The most immediate step is to show the potential and establish relationships between Code and Math. This will serve as a precedent to start building curriculum as a new way to teach.

framework

The following examples showcase how Code Interacts with Math to show relationships and build upon a goal. 12 exercises that walk the student from Variables to graphing in the Cartisean Plane.
